# CareConnect Desktop — Keyboard Shortcuts

All primary actions are keyboard-accessible. On macOS, substitute `Cmd` for `Ctrl`.

## File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Create new task |
| `Ctrl + S` | Save current changes |
| `Ctrl + Q` | Quit application |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + 1` | Go to Dashboard |
| `Ctrl + 2` | Go to Tasks |
| `Ctrl + 3` | Go to Calendar |
| `Ctrl + 4` | Go to Settings |
| `Tab` | Move focus to next element |
| `Shift + Tab` | Move focus to previous element |
| `Enter` | Activate focused button or link |

## Search & Edit

| Shortcut | Action |
|----------|--------|
| `Ctrl + F` | Focus search bar |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `Ctrl + A` | Select all |
| `Ctrl + C` | Copy |
| `Ctrl + V` | Paste |

## Accessibility

| Shortcut | Action |
|----------|--------|
| `Ctrl + =` | Increase font size |
| `Ctrl + -` | Decrease font size |
| `Ctrl + Shift + H` | Toggle high contrast mode |
| `Ctrl + Shift + D` | Toggle dark mode |

## Dialogs

| Shortcut | Action |
|----------|--------|
| `Escape` | Close current dialog |
| `F1` | Show keyboard shortcuts reference |

## Task List

| Shortcut | Action |
|----------|--------|
| `Enter` | Open selected task detail |
| `Space` | Toggle task completion |
